# PHP CRUD Project

This is a simple PHP CRUD application with Admin panel.

## Features
- User Login & Register
- Admin role
- Add / Edit / Delete Posts
- Image Upload
- MySQL Database

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- XAMPP

## How to Run
1. Install XAMPP
2. Copy project to htdocs folder
3. Import database.sql in phpMyAdmin
4. Open browser and run:
   http://localhost/crud/posts.php

## Admin Login
Create an admin user manually in database:
